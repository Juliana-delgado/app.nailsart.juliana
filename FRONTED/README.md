# Stitch Nail Salon — Frontend

Proyecto frontend minimal para el Sistema de Gestión del Salón de Uñas.

Instalación y ejecución:

```bash
cd FRONTED
npm install
npm run dev
```

Características:
- Estructura MVP (models, presenters, views)
- Login (usuario por defecto: admin/admin)
- Gestión de Clientes con almacenamiento en `localStorage`
- Plantilla para Diseños, Turnos y Promociones
